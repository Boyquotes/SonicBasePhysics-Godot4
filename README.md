# SonicBasePhysics-Godot4
 A simple base for a Sonic-like game for Godot 4 (beta 8) by ChrisFurry
 # Features
* 360 Sonic Physics
* Basic States
	* Normal Movement
	* Jumping
	* Rolling
* Layer Switching
* LOOPS!!!!!!!!!!
